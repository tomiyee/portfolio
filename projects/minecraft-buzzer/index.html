Minecraft, but It Beeps in Pain
Aug 9, 2020 (Source Code)

Minecraft Damage Buzzer Plugin!

Have you ever wanted to hear your Minecraft character scream whenever it takes damage? Well say no more! I've made a piezoelectric buzzer scream everytime you get hurt! Check out the demo below!



This Minecraft Java Plugin I made sends POST requests to a node.js web server hosted on a Raspberry Pi 4 whenever a player takes damage. The web server counts the amount of damage taken and serves it on a GET request. An ESP32 with a buzzer is sending repeated GET requests to the same server waiting for this number to change. The ESp32 will acivate the buzzer briefly whenever a change is detected.
