Hooked.io
July 2020 (Source Code)

Our Hooked.io Web Game

This game is a collaborative effort between myself and Isabella Kang. Our game, Hooked.io, is inspired by similar web browser games in the io-genre, which are multiplayer adversarial games. Popular examples of io-games include agar.io and snake.io. Below this introduction, you should see the game screen. (If you would like to open this game in a separate tab, click on this link.)

To begin the game, enter the username you would like to appear with your avatar and select play. You move in the direction of your cursor. The shadowy circles are fish, which you catch with your hook. To fire your hook and catch fish, you left-click with your cursor. If there are other players, you can hook them in as well as long as your ship is larger.

I would highly recommend checking the Readme in the Github Repository for a more detailed description of the mechanics.

To demonstrate the multiplayer capability, you could open this page in another tab and enter a different name. You would see that both "players" can be seen in the leaderboards in the top right.
