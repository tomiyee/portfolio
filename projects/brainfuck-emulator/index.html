Brainfuck Emulator
Mar 6, 2019 (Source Code)
How to Use

The top text area is where you enter the brainfuck code. The bottom window is the "Terminal," where the code's output can be seen and inputs can be entered. When you enter inputs, only the first character is considered with 3 exceptions.

If the first two characters of the input are 'b, 'h, or 'd, the rest of the input will be interpretted as a binary, hexadecimal, or decimal number respectively.

You can see the values in memory by clicking on the [See Memory] button below. If you keep the Cell Values popup open while the program runs, you can see the memory change as the program runs.


  • Run Code: Shift + Enter
  • Save Code: Ctrl + S
  • Interrupt: Ctrl + C
